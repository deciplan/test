🔐 Deciplan Encoder

Outil de chiffrement pour mettre à jour les données d'un étudiant

⚠️ Confidentialité : Cet outil fonctionne 100% dans votre navigateur. Les données ne sont jamais envoyées sur internet — elles ne quittent pas votre ordinateur.

1Coller le JSON en clair

Ouvrez votre fichier {prenom-nom}-plain.json avec un éditeur de texte, copiez tout son contenu et collez-le ici.

2Informations de l'étudiant

L'identifiant est utilisé dans l'URL personnelle de l'étudiant (ex: deciplan.com/?e=fatima-farhat). La CIN est la clé de déverrouillage.

3Chiffrer et télécharger

Le fichier sera téléchargé sous le nom {identifiant}.json. Uploadez-le sur Hostinger dans le dossier eleves/ de votre site.

💡 Astuce : Gardez les fichiers {prenom-nom}-plain.json uniquement sur votre ordinateur.
Seuls les fichiers chiffrés du dossier eleves/ doivent être uploadés sur Hostinger.
